📌 1. Why Loops? The Power of Repetition

A programmer writes code once and if repetition is needed, the program repeats it automatically — not by copy-pasting. Loops allow us to execute a block of code multiple times. Common use cases: array manipulation, DOM traversal, event handling, animations, and data processing.

💡 Without loops, printing 500 stars would require 500 console.log statements. With a loop, just 4 lines of code!

In JavaScript, we have: for loop, while loop, do-while loop, for-of loop, for-in loop. Today we focus on for, while, and do-while.

📌 తెలుగు : ఒక programmer code ని ఒక్కసారి రాస్తాడు, repetition అవసరమైతే program దాన్ని తనంతట తానే repeat చేసుకుంటుంది — copy-paste కాదు. Loops వలన ఒక block ని multiple times execute చేయవచ్చు. ఉదా: 500 stars print చేయడానికి 500 console.log రాయాల్సిన అవసరం లేదు; ఒక్క loop చాలు.

⭐ 2. The Problem: Printing 5 stars vertically vs horizontally

console.log() always adds a new line after printing. So multiple console.log statements print stars one below another (vertically).

console.log("*"); // star + newline console.log("*"); // another star below // Output: // * // *

To print stars horizontally (one beside another), we need a different approach: build a string and print once, or use process.stdout.write in Node.js. But for pattern programming, we will first understand loop mechanics using vertical stars.

📌 తెలుగు : console.log() ప్రతిసారీ new line వేస్తుంది. కాబట్టి multiple console.log statements stars ని ఒకదాని కింద ఒకటి print చేస్తాయి (vertical). Horizontal గా print చేయడానికి ప్రత్యేక పద్ధతి ఉంది. ముందు loop mechanics ని vertical stars తో అర్థం చేసుకుందాం.

🔁 3. For Loop — When you know number of iterations

Every loop has 4 components:

① Initialization ② Condition ③ Body ④ Increment/Decrement
// Print 5 stars (vertical) using for loop for (let i = 1; i <= 5; i++) { console.log("*"); } // Output: // * // * // * // * // *

Execution flow: Initialize (i=1) → Check condition (1<=5 true) → Execute body (print star) → Increment (i becomes 2) → Check condition (2<=5 true) → Execute body → ... until i=6, condition fails → loop stops.

✅ For loop is ideal when you know exactly how many times to repeat (e.g., print 10 stars, process array of known length).

📌 తెలుగు : ప్రతీ loop లో 4 components: Initialization, Condition, Body, Increment/Decrement. for (let i=1; i<=5; i++) { console.log("*"); } — ఇది 5 stars vertical గా print చేస్తుంది. For loop ఎప్పుడు వాడతాం? మనకు ఎన్నిసార్లు repeat చేయాలో తెలిసినప్పుడు (ఉదా: 10 stars, array length).

🔄 4. While Loop — When iterations are unknown

While loop checks the condition before executing the body. If condition is false initially, body never executes.

let i = 1; while (i <= 5) { console.log("*"); i++; } // Output: 5 stars vertically

Execution: Initialize outside → Check condition → if true, execute body → increment → check condition again → repeat until condition false.

💡 Use while loop when you don't know the exact number of iterations, but have a condition (e.g., keep taking user input until they type "quit").

📌 తెలుగు : While loop condition before body execute అవుతుంది. Initialization outside చేయాలి. ఎప్పుడు వాడతాం? ఎన్నిసార్లు repeat చేయాలో తెలియనప్పుడు, కానీ ఒక condition మీద ఆధారపడి ఉన్నప్పుడు (ఉదా: user "quit" అనే వరకు input తీసుకోవడం).

🔂 5. Do-While Loop — Execute first, check later

Do-while loop executes the body at least once, then checks the condition. Useful when the action must happen before evaluation.

let i = 1; do { console.log("*"); i++; } while (i <= 5); // Output: 5 stars vertically

Difference: Even if condition is false initially, do-while runs once. While loop would skip entirely.

let j = 10; while (j <= 5) { // condition false → never runs console.log("while"); } do { // runs once even though condition false console.log("do-while runs at least once"); } while (j <= 5);

📌 తెలుగు : Do-while loop body ని కనీసం ఒక్కసారి execute చేస్తుంది, తర్వాత condition check చేస్తుంది. ఇది while loop కంటే భిన్నం — while లో condition false అయితే body execute అవ్వదు. ఉదా: user input తీసుకోవడానికి ముందు prompt చూపించాల్సిన సందర్భాలలో do-while ఉపయోగపడుతుంది.

⚖️ 6. Comparison: For vs While vs Do-While
// SAME OUTPUT using different loops // For loop for (let i = 1; i <= 5; i++) { console.log("*"); } // While loop let i = 1; while (i <= 5) { console.log("*"); i++; } // Do-while loop let j = 1; do { console.log("*"); j++; } while (j <= 5);
🎯 When to use which?
• For: Known number of iterations (arrays, fixed counts).
• While: Unknown iterations but condition-based (user input, reading files until EOF).
• Do-While: Must execute at least once (menu-driven programs, retry prompts).

There are also for-of (iterating arrays/strings) and for-in (iterating object properties) — we will learn them after covering arrays and objects.

📌 తెలుగు : మూడు loops ఒకే output ఇవ్వగలవు. ఎప్పుడు ఏది వాడాలి?
• for: ఎన్నిసార్లు repeat చేయాలో తెలిస్తే (ఉదా: 10 stars).
• while: ఎన్నిసార్లు repeat అవ్వాలో తెలియకపోయినా, ఒక condition true ఉన్నంత వరకు repeat అవ్వాల్సినప్పుడు.
• do-while: కనీసం ఒక్కసారి అయినా execute అవ్వాల్సినప్పుడు (ఉదా: menu driven program).

🎨 7. Introduction to Pattern Programming (Next Steps)

Pattern programming (printing stars in shapes like triangles, rectangles, pyramids) is the best way to master nested loops. For example:

// Rectangle of 5x5 stars for (let row = 1; row <= 5; row++) { let line = ""; for (let col = 1; col <= 5; col++) { line += "*"; } console.log(line); } // Output: // ***** // ***** // ***** // ***** // *****
🔥 Kalyan's Advice: Solve at least 200-250 pattern problems to build logical thinking. Loops are the heart of programming — practice with varying conditions (<=, >=, different increments).

In upcoming classes, we will solve triangle patterns, pyramid patterns, number patterns, and more — solidifying your loop mastery.
