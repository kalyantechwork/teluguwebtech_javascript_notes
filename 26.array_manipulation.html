🛠️ Array Manipulation in JavaScript

📦 push, pop, shift, unshift, splice (add/remove/replace), indexOf, lastIndexOf, includes — by Kalyan (Telugu WebTech)
➕ 1. Adding Elements: push() and unshift()
let arr = [10, 20, 30, 40, 50]; // Add to the END arr.push(60); console.log(arr); // [10, 20, 30, 40, 50, 60] // Add to the BEGINNING arr.unshift(70); console.log(arr); // [70, 10, 20, 30, 40, 50, 60]
✅ push() — adds element(s) at the end (efficient, O(1) complexity).
✅ unshift() — adds element(s) at the beginning (costly, shifts all elements right).
⚠️ Costly operation: unshift() shifts every existing element to the right → slower for large arrays.

📌 తెలుగు : push() — end లో add చేస్తుంది (efficient). unshift() — beginning లో add చేస్తుంది (costly — అన్ని elements right shift అవుతాయి).

➖ 2. Removing Elements: pop() and shift()
let arr = [10, 20, 30, 40, 50]; // Remove from the END (returns removed element) let last = arr.pop(); console.log(last); // 50 console.log(arr); // [10, 20, 30, 40] // Remove from the BEGINNING (returns removed element) let first = arr.shift(); console.log(first); // 10 console.log(arr); // [20, 30, 40]
✅ pop() — removes last element (efficient, O(1)).
✅ shift() — removes first element (costly, shifts all elements left).
⚠️ Both methods return the removed element and modify the original array.

📌 తెలుగు : pop() — చివరి element ని తొలగిస్తుంది (efficient). shift() — మొదటి element ని తొలగిస్తుంది (costly — remaining elements left shift). రెండూ తొలగించిన element ని return చేస్తాయి.

🔧 3. splice() - Add, Remove, Replace at Any Position

splice(start, deleteCount, item1, item2, ...) — modifies original array.

let arr = [10, 20, 30, 40, 50]; // 1. Remove 1 element at index 2 let removed = arr.splice(2, 1); console.log(removed); // [30] console.log(arr); // [10, 20, 40, 50] // Reset array arr = [10, 20, 30, 40, 50]; // 2. Remove 2 elements starting at index 2 removed = arr.splice(2, 2); console.log(removed); // [30, 40] console.log(arr); // [10, 20, 50] // 3. Insert (deleteCount = 0) arr = [10, 20, 30, 40, 50]; arr.splice(2, 0, 60); // insert 60 at index 2 console.log(arr); // [10, 20, 60, 30, 40, 50] // 4. Replace: delete 1, insert 2 items arr = [10, 20, 30, 40, 50]; arr.splice(2, 1, 60, 70); console.log(arr); // [10, 20, 60, 70, 40, 50]
💡 splice() return value: An array containing the deleted elements (empty array if nothing deleted).

📌 తెలుగు : splice(start, deleteCount, items...) — array ని modify చేస్తుంది. deleteCount=0 అయితే insert చేస్తుంది. deleteCount >0 అయితే remove చేస్తుంది. items pass చేస్తే replace/insert చేస్తుంది. ఇది చాలా powerful method.

🔍 4. Searching: indexOf(), lastIndexOf(), includes()
let arr = [10, 20, 30, 40, 50, 30]; // Find first occurrence console.log(arr.indexOf(30)); // 2 (first 30) // Find last occurrence console.log(arr.lastIndexOf(30)); // 5 (last 30) // Check if element exists console.log(arr.includes(40)); // true console.log(arr.includes(100)); // false // When element not found, indexOf returns -1 console.log(arr.indexOf(100)); // -1
✅ indexOf(element) — returns first index or -1 if not found.
✅ lastIndexOf(element) — returns last index or -1.
✅ includes(element) — returns true/false.

📌 తెలుగు : indexOf(element) — మొదటి occurrence index (లేకపోతే -1). lastIndexOf(element) — చివరి occurrence index. includes(element) — true/false (element ఉందో లేదో).

⏱️ 5. Efficiency Note: Which Operations are Costly?
  • push() / pop() → Fast (O(1)) — operate at the end.
  • unshift() / shift() → Slow (O(n)) — require shifting all elements.
  • splice() → O(n) in worst case (shifting elements).
💡 For large arrays, prefer push/pop over unshift/shift to maintain performance.
// Good for performance arr.push(100); // fast arr.pop(); // fast // Avoid for large arrays if possible arr.unshift(0); // slow (shifts all) arr.shift(); // slow (shifts all)

📌 తెలుగు : push/pop → fast (O(1)). unshift/shift → slow (O(n)) — all elements shift. పెద్ద arrays కోసం push/pop ని prefer చేయండి.
