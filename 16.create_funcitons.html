📝 5 Ways to Write Functions in JavaScript

🔧 Function Declaration | Function Expression | Arrow Function | IIFE | Generator Function — by Kalyan (Telugu WebTech)
📌 1. Overview: Five Different Ways to Create Functions

In JavaScript, functions are first-class citizens. There are five main ways to write/create functions:

  • Function Declaration — most common, traditional way
  • Function Expression — assigning an anonymous function to a variable
  • Arrow Function — modern concise syntax (ES6)
  • IIFE (Immediately Invoked Function Expression) — function that runs as soon as it's defined
  • Generator Function — special function that can pause and resume (advanced, covered later)
💡 Understanding all these forms will help you read and write modern JavaScript code with confidence.

📌 తెలుగు : JavaScript లో functions ని 5 విధాలుగా రాయవచ్చు:
1. Function Declaration — సాధారణ, traditional way
2. Function Expression — anonymous function ని variable కి assign చేయడం
3. Arrow Function — concise syntax (ES6)
4. IIFE — define అయిన వెంటనే execute అయ్యే function
5. Generator Function — advanced (pause/resume చేయగల function)

1 Function Declaration

The most traditional way. Uses the function keyword, a name, parameters, and a body.

// Function Declaration function add(a, b) { return a + b; } console.log(add(10, 20)); // Output: 30
✅ Hoisting: Function declarations are hoisted — they can be called before they are defined in the code.

📌 తెలుగు : Function Declaration: function keyword తో రాస్తారు. ఇవి hoisted అవుతాయి — define చేయడానికి ముందే call చేయవచ్చు. ఉదా: function add(a,b){ return a+b; }

2 Function Expression

An anonymous function (no name) is created and assigned to a variable. The function is called using the variable name.

// Function Expression let add = function(a, b) { return a + b; }; console.log(add(10, 20)); // Output: 30
✅ Unlike declarations, function expressions are not hoisted. You can only call them after the assignment.

📌 తెలుగు : Function Expression: Anonymous function ని variable కి assign చేస్తారు. ఇవి hoisted కావు — define చేసిన తర్వాత మాత్రమే call చేయాలి. ఉదా: let add = function(a,b){ return a+b; }

3 Arrow Function (Modern ES6)

Introduced in ES6 (2015). Shorter syntax using => (arrow). No function keyword. Works great for callbacks and simple logic.

// Arrow function - full form let add = (a, b) => { return a + b; }; // Arrow function - concise (one-liner) - implicit return let addConcise = (a, b) => a + b; console.log(add(50, 60)); // Output: 110 console.log(addConcise(50, 60)); // 110
🔥 Note: If the body has only one statement that returns a value, you can omit {} and return. This is called implicit return.

For beginners: Start with the full syntax (with { return ... }) for clarity. As you gain experience, you can use the concise form.

📌 తెలుగు : Arrow Function (ES6): => సింటాక్స్. function keyword అవసరం లేదు. ఒకే statement ఉంటే {} మరియు return ని తొలగించవచ్చు (implicit return). ఉదా: let add = (a,b) => a + b; మొదల్లో full syntax వాడండి, అనుభవం వచ్చాక concise form వాడండి.

4 IIFE (Immediately Invoked Function Expression)

An IIFE is a function that runs immediately after it is defined. No need to call it separately. It is a function expression wrapped in parentheses and then invoked with ().

// IIFE - runs immediately (function() { console.log("I run immediately!"); })(); // IIFE with parameters (function(name) { console.log("Hello, " + name); })("Kalyan"); // Arrow function IIFE (() => { console.log("Arrow IIFE"); })();
💡 Use cases: Creating a private scope, avoiding global variable pollution, one-time initialization code.

Key syntax: Wrap the function expression in ( ) and then add ( ) to invoke immediately.

📌 తెలుగు : IIFE (Immediately Invoked Function Expression): Define అయిన వెంటనే execute అయ్యే function. దీనికి ప్రత్యేకంగా call చేయాల్సిన అవసరం లేదు. Syntax: (function(){ ... })();. Use cases: private scope, global namespace pollution ని avoid చేయడం, one-time initialization.

5 Generator Function (Advanced)

Generator functions can be paused and resumed. They use function* syntax and the yield keyword. They return a special iterator object. We will explore them in detail when we cover advanced JavaScript and asynchronous patterns.

// Generator function (simple preview) function* generateNumbers() { yield 1; yield 2; yield 3; } const gen = generateNumbers(); console.log(gen.next().value); // 1 console.log(gen.next().value); // 2
🔮 Coming later: Generator functions will be covered in depth during object-oriented JavaScript and advanced async topics.

For now, focus on mastering function declarations, expressions, arrow functions, and IIFE.

📌 తెలుగు : Generator Function: function* సింటాక్స్, yield keyword. ఇవి pause/resume అవ్వగల functions. Advanced concepts లో వివరంగా చూద్దాం. ప్రస్తుతానికి Declaration, Expression, Arrow, IIFE లను బాగా practice చేయండి.

📊 6. Quick Comparison: Declaration vs Expression vs Arrow vs IIFE
// 1. Function Declaration function declareAdd(a, b) { return a + b; } // 2. Function Expression const expressAdd = function(a, b) { return a + b; }; // 3. Arrow Function const arrowAdd = (a, b) => a + b; // 4. IIFE (function() { console.log("IIFE runs immediately"); })(); // All work the same way when called console.log(declareAdd(5, 3)); // 8 console.log(expressAdd(5, 3)); // 8 console.log(arrowAdd(5, 3)); // 8
🧠 Kalyan's advice:
  • Start with Function Declaration — it's the most readable for beginners.
  • Use Arrow Functions for short callbacks and simple logic.
  • Use IIFE when you need a one-time execution with private scope.
  • Generator functions — master the basics first, then learn them.

📌 తెలుగు సారాంశం :
• Function Declaration: చాలా clear, beginners కి best.
• Function Expression: variable కి assign చేస్తారు.
• Arrow Function: short syntax, callbacks కి perfect.
• IIFE: immediately execute, private scope కోసం.
• Generator: advanced, తర్వాత నేర్చుకోండి.
