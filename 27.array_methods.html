map() Transform Each Element → New Array

map() creates a new array by applying a callback function to each element of the original array. The new array has the same length.

let numbers = [1, 4, 9, 16]; // Calculate square root of each element let result = numbers.map(x => Math.sqrt(x)); console.log(result); // [1, 2, 3, 4] // Original array unchanged console.log(numbers); // [1, 4, 9, 16]
✅ Syntax: arr.map(callback(currentValue, index, array))
💡 Use case: Convert data (e.g., array of numbers to strings, extract property from objects, apply math operations).
// More examples let prices = [100, 200, 300]; let withTax = prices.map(price => price * 1.18); console.log(withTax); // [118, 236, 354]

📌 తెలుగు : map() ప్రతీ element పై callback ని apply చేసి కొత్త array ని create చేస్తుంది (length same). ఉదా: square roots, tax calculation. Original array మారదు.

filter() Keep Elements That Pass a Condition

filter() creates a new array with only elements that satisfy a condition (callback returns true).

let ages = [32, 15, 19, 12]; // Keep only ages >= 18 let adults = ages.filter(age => age >= 18); console.log(adults); // [32, 19] // Original array unchanged console.log(ages); // [32, 15, 19, 12]
✅ Callback must return true/false. True → include element, False → exclude.
let scores = [85, 67, 72, 90]; let above80 = scores.filter(score => score > 80); console.log(above80); // [85, 90]

📌 తెలుగు : filter() condition ని satisfy చేసే elements తో కొత్త array ని create చేస్తుంది. Callback true return చేస్తే element retain అవుతుంది; false అయితే exclude అవుతుంది.

reduce() Aggregate Array into a Single Value

reduce() reduces the array to a single value (sum, product, concatenation, etc.). It takes a callback and an initial value.

let numbers = [1, 4, 9, 16]; // Sum of all elements let sum = numbers.reduce((total, current) => total + current, 0); console.log(sum); // 30 (1+4+9+16) // Product of all elements let product = numbers.reduce((total, current) => total * current, 1); console.log(product); // 576 (1*4*9*16)
✅ Syntax: arr.reduce(callback(accumulator, currentValue), initialValue)
💡 Use cases: Sum, average, max/min, flatten arrays, etc.

📌 తెలుగు : reduce() array ని ఒక single value గా మారుస్తుంది (sum, product). Callback రెండు arguments తీసుకుంటుంది: accumulator (aggregator) మరియు currentValue. Initial value తప్పనిసరిగా ఇవ్వాలి.

some() & every() Check Conditions on Array
let scores = [85, 67, 72, 90]; // some(): at least one element satisfies condition let hasGradeA = scores.some(score => score > 85); console.log(hasGradeA); // true (90 > 85) // every(): ALL elements must satisfy condition let allPassed = scores.every(score => score >= 50); console.log(allPassed); // true (all >= 50) let allAbove80 = scores.every(score => score > 80); console.log(allAbove80); // false (67 and 72 are not >80)
✅ some() → returns true if AT LEAST ONE element passes.
✅ every() → returns true if EVERY element passes.

📌 తెలుగు : some() — కనీసం ఒక్క element అయినా condition satisfy చేస్తే true. every() — అన్ని elements satisfy చేస్తేనే true.

find() & findIndex() Locate First Matching Element
let scores = [85, 67, 72, 90]; // find(): returns the first element that satisfies condition let firstAbove70 = scores.find(score => score > 70); console.log(firstAbove70); // 85 (first element >70) // findIndex(): returns index of first element that satisfies condition let indexAbove70 = scores.findIndex(score => score > 70); console.log(indexAbove70); // 0 (index of 85) // When no match found let above95 = scores.find(score => score > 95); console.log(above95); // undefined
✅ find() → returns the element (or undefined).
✅ findIndex() → returns index (or -1 if not found).

📌 తెలుగు : find() — మొదటి matching element ని return చేస్తుంది (లేకపోతే undefined). findIndex() — matching element యొక్క index ని return చేస్తుంది (లేకపోతే -1).

sort() Sorting Arrays
// String sorting (default - alphabetical) let fruits = ["Banana", "Apple", "Pear", "Kiwi"]; fruits.sort(); console.log(fruits); // ["Apple", "Banana", "Kiwi", "Pear"] // Number sorting (default converts to strings → gives incorrect order!) let scores = [85, 67, 72, 90]; scores.sort(); console.log(scores); // [67, 72, 85, 90] (works here by chance) // For numbers, use compare callback let numbers = [10, 2, 30, 5]; numbers.sort((a, b) => a - b); // ascending console.log(numbers); // [2, 5, 10, 30] numbers.sort((a, b) => b - a); // descending console.log(numbers); // [30, 10, 5, 2]
🔥 Important: Default sort converts elements to strings. For numbers, always provide a compare function: (a, b) => a - b for ascending.

📌 తెలుగు : sort() default గా strings గా convert చేసి sort చేస్తుంది. Numbers కోసం compare callback ఇవ్వాలి: (a,b) => a - b (ascending) లేదా b - a (descending).

📖 7. Quick Reference: Array Methods with Callbacks
let arr = [1, 2, 3, 4, 5]; // map: transform each let doubled = arr.map(x => x * 2); // [2,4,6,8,10] // filter: keep condition let evens = arr.filter(x => x % 2 === 0); // [2,4] // reduce: aggregate let sum = arr.reduce((acc, x) => acc + x, 0); // 15 // some: any element satisfies? let hasEven = arr.some(x => x % 2 === 0); // true // every: all elements satisfy? let allPositive = arr.every(x => x > 0); // true // find: first element satisfying let firstEven = arr.find(x => x % 2 === 0); // 2 // findIndex: index of first matching let indexEven = arr.findIndex(x => x % 2 === 0); // 1

• reduce → array ని single value గా మారుస్తుంది.
