🧠 1. Memory Inside RAM: Stack & Heap

When any program (including JavaScript) executes, it resides inside the RAM. Internally, JavaScript automatically divides the RAM into two partitions: Stack and Heap. Stack stores simple values (primitives) and references, while Heap stores complex objects. Understanding this memory model makes you a next-level programmer.

✅ Variables are like small boxes inside Stack memory. Each variable has a name (identifier) and stores a value (like 23, "hello", true).
// Variable declaration + assignment (syntax) let age = 23; // 'let' keyword creates variable in stack console.log(age); // prints 23

We will explore Stack and Heap in depth throughout the course. For now, remember: every variable lives in stack, and data types define how values are stored and converted to binary.

📌 తెలుగు : JavaScript program execute అయినప్పుడు అది RAM లోనే ఉంటుంది. JavaScript RAM ని రెండు భాగాలుగా విభజిస్తుంది: Stack మరియు Heap. Stack లో simple values (primitives) నిల్వ అవుతాయి. Variable అనేది stack లోని ఒక చిన్న box లాంటిది. ఉదా: let age = 23; — age అనే variable stack లో 23 ని store చేస్తుంది. ఈ memory view వలన data types ఎలా పనిచేస్తాయో స్పష్టమవుతుంది.

🔢 2. Number Data Type (Integers, Floats, Infinity, NaN)

Number in JavaScript covers both integers (whole numbers like 23, 2000) and floating-point numbers (decimals like 5.9, 68.2). There's no separate 'int' or 'float'. JavaScript uses a single Number type.

let age = 23; // integer let height = 5.9; // float (real number) let weight = 68.2; console.log(age, height, weight);

Special numeric values: Infinity, -Infinity, and NaN (Not a Number).

let positiveInf = 1 / 0; // Infinity let negativeInf = -1 / 0; // -Infinity let notNumber = "hello" / 2; // NaN console.log(typeof positiveInf); // "number"

Use typeof operator to check the data type: typeof 42 → "number".

💡 JavaScript is loosely typed (dynamic): you don't declare type. The same variable can hold number, then string, etc. The engine decides the type automatically.

📌 తెలుగు : JavaScript లో Number data type integers (23, 2000) మరియు floating-point numbers (5.9, 68.2) రెండింటినీ కవర్ చేస్తుంది. ప్రత్యేకమైన int, float లేవు. Special numeric values: Infinity (1/0), -Infinity, NaN (Not a Number). typeof ఆపరేటర్ type ని తెలుసుకోవడానికి వాడతాం. JavaScript loosely typed language — variable ఏ type అయినా store చేసుకోగలదు.

📊 3. Range of Number & Introduction to BigInt

Every data type has a range (minimum to maximum value it can store). In JavaScript, the Number type can safely store integers up to 2⁵³ - 1 (approx 9 quadrillion). Beyond that, precision is lost.

let maxNumber = 9007199254740991; // 2⁵³ - 1 console.log(maxNumber); // works fine let beyondMax = 9007199254740992; // loss of precision possible

To store very large integers beyond this range, JavaScript provides BigInt. Append n at the end of an integer to create a BigInt.

let hugeNumber = 123456789012345678901234567890n; // BigInt console.log(typeof hugeNumber); // "bigint" let another = 100n; // arithmetic between two BigInts works console.log(hugeNumber + another);
✅ BigInt rules: Only for integers (no decimals). Cannot mix BigInt with regular Number in arithmetic operations (error). Use BigInt when you need arbitrarily large integers.

Suffix 'n' makes a number a BigInt. Example: let big = 9999999999999999n;. Never use decimal with BigInt (like 10.5n → error).

📌 తెలుగు : Number data type range: 2⁵³ - 1 వరకు safe integers. దీనికి మించిన పెద్ద integers ని store చేయడానికి BigInt ఉపయోగిస్తాం. BigInt కోసం సంఖ్య చివర n suffix జోడించాలి. ఉదా: let x = 9876543210987654321n;. BigInt లో decimals పనిచేయవు. BigInt మరియు Number ని mix చేసి arithmetic చేయడం వలన error వస్తుంది.

📝 4. String & Boolean Data Types

String: A series of characters (text) enclosed in double quotes, single quotes, or backticks. Examples: "Tap Academy", 'Kalyan', `Hello`.

let institute = "Tap Academy"; console.log(institute); // Tap Academy console.log(typeof institute); // "string"

Boolean: Represents logical values — only true or false. Commonly used for conditions, flags (hasPassport, isLoggedIn).

let hasPassport = true; let isPlaced = false; console.log(typeof hasPassport); // "boolean"

Booleans are the result of comparisons: 10 > 5 → true, 3 === '3' → false.

📌 తెలుగు : String: అనేది characters యొక్క సరళి. ఉదా: "నమస్కారం", 'Kalyan'. Boolean: కేవలం true లేదా false విలువలను మాత్రమే సూచిస్తుంది. ఉదా: let hasPassport = false;. typeof boolean gives "boolean".

❓ 5. Undefined: Variable declared but no value assigned

When you declare a variable using let or var but do not assign a value, JavaScript automatically assigns the value undefined. Both its value and type are 'undefined'.

let a; // no assignment console.log(a); // undefined console.log(typeof a); // "undefined"
💡 Important: Undefined means the variable exists in memory (stack) but hasn't been given a value yet. It's different from "not defined" (which throws an error).

We will explore undefined deeper with the concept of hoisting in future classes.

📌 తెలుగు : ఒక variable ని declare చేసి value assign చేయకపోతే, JavaScript దానికి undefined అనే value ని ఇస్తుంది. ఉదా: let x; console.log(x); → undefined. దాని type కూడా "undefined" అనే వస్తుంది. ఇది "not defined" error కంటే భిన్నం.

⚡ 6. typeof Operator & Loosely Typed Nature

typeof operator returns the type of a variable or value. Since JavaScript is dynamically typed, the same variable can hold different types over time.

let data = 100; console.log(typeof data); // "number" data = "Now I am a string"; console.log(typeof data); // "string" data = true; console.log(typeof data); // "boolean"

This flexibility is powerful but also requires careful handling. Unlike strongly typed languages (Java, C++), you don't specify types like int or float. JavaScript handles conversion automatically.

✅ Summary of data types covered: Number, BigInt, String, Boolean, Undefined. (Null and Symbol will be covered in object-oriented sections later.)

Special numeric behaviors: typeof NaN → "number", typeof Infinity → "number".

📌 తెలుగు : typeof operator variable యొక్క type ని తెలుపుతుంది. JavaScript dynamic typing వలన ఒకే variable లో number, string, boolean వంటి విభిన్న types ని store చేయవచ్చు. ఉదా: let x = 42; x = "hello"; — type number నుండి string కి మారింది. ఇది loose typing యొక్క ప్రత్యేకత.
