🔄 1. What is Type Conversion (Type Casting)?

In JavaScript, we often need to convert one data type to another. For example: converting a Number to String, Boolean to String, String to Number, or Boolean to Number. This process is called type conversion or type casting. JavaScript provides built-in functions like String(), Number(), Boolean(), and parseInt() for this purpose.

✅ Type conversion = converting a value from one data type to another (e.g., number → string, string → number, etc.)

Understanding how type conversion works is crucial for technical interviews and writing bug-free code.

📌 తెలుగు : JavaScript లో మనం ఒక data type ని మరొక data type గా మార్చవలసి ఉంటుంది. ఉదా: Number ని String గా, Boolean ని String గా, String ని Number గా. దీన్నే type conversion లేదా type casting అంటారు. దీనికి String(), Number(), Boolean(), parseInt() వంటి built-in functions ఉన్నాయి.

🔢 2. Number → String using String() & Memory Insight

Convert number to string: Use String(value).

let a = 123; // number console.log(a, typeof a); // 123 "number" let b = String(a); // converts COPY of a's value to string console.log(b, typeof b); // "123" "string"

Important memory concept: When you convert, JavaScript creates a copy of the value and converts that copy. The original variable a remains unchanged. Always store the converted result in a new variable if you need it later.

💡 Type conversion does NOT modify the original variable. It returns a new converted value.

Same works for Boolean → String: String(true) → "true", String(false) → "false".

📌 తెలుగు : Number ని String గా మార్చడానికి String(value) ఉపయోగిస్తాం. ఇది original variable ని మార్చదు, copy ని convert చేస్తుంది. Convert చేసిన value ని store చేయడానికి కొత్త variable లో ఉంచాలి. Boolean → String కూడా అలాగే పనిచేస్తుంది: String(true) → "true".

🔢 3. String → Number: Number() vs parseInt()

Method 1: Number() — tries to convert entire string to number. If the string contains any non-numeric character (except leading whitespace), it returns NaN.

Number("123") // 123 Number("123abc") // NaN Number(" 42 ") // 42 (whitespace allowed)

Method 2: parseInt() — reads from left to right, extracts integer until first non-numeric character, then stops. Very useful for strings like "123abc".

parseInt("123abc") // 123 (stops at 'a') parseInt("abc123") // NaN (first char not numeric) parseInt(" 123abc") // 123 (whitespace allowed before) parseInt("123 456") // 123 (stops at space)
✅ parseInt() logic: Starts from left, skips whitespace, takes consecutive numeric characters, stops at first non-digit. Returns NaN if no valid number found.

For decimal numbers, use parseFloat().

📌 తెలుగు : String ని Number లోకి మార్చడానికి రెండు విధానాలు:
• Number(): పూర్తి string ని number గా మార్చడానికి ప్రయత్నిస్తుంది. అక్షరాలు (a,b, etc) వస్తే NaN ఇస్తుంది.
• parseInt(): left నుండి right కి scan చేస్తుంది, మొదటి non-numeric character దగ్గర ఆగిపోతుంది. "123abc" → 123 ఇస్తుంది. మొదటి character non-numeric అయితే NaN వస్తుంది.

✅ 4. Boolean → Number (true → 1, false → 0)

Converting Boolean to Number is straightforward: Number(true) → 1, Number(false) → 0.

let a = true; let b = Number(a); // 1 console.log(b, typeof b); // 1 "number" let c = false; let d = Number(c); // 0

This is very useful in mathematical operations and conditional logic.

📌 తెలుగు : Boolean ని Number లోకి మార్చడం చాలా simple: Number(true) → 1, Number(false) → 0. గుర్తుంచుకోండి: true = 1, false = 0.

🎯 5. Number & String → Boolean: Truthy and Falsy values

Use Boolean(value) to convert any value to true/false.

Falsy values (convert to false):

0 NaN "" (empty string) null undefined false

Truthy values (convert to true): Everything else — any non-zero number (positive/negative), any non-empty string (including "0", "false", " "), Infinity, -Infinity, arrays, objects, etc.

Boolean(123) // true Boolean(-5) // true Boolean(0) // false Boolean("hello") // true Boolean("") // false Boolean(" ") // true (space is a character) Boolean(NaN) // false Boolean(Infinity) // true
🔥 Interview favorite: Truthy = any value that converts to true. Falsy = only 0, NaN, "", null, undefined, false. Memorize these!

📌 తెలుగు : Boolean(value) ని ఉపయోగించి ఏ value నైనా true/false లోకి మార్చవచ్చు.
Falsy values (false గా మారేవి): 0, NaN, "" (empty string), null, undefined, false.
Truthy values: మిగిలినవన్నీ — non-zero numbers, non-empty strings, Infinity, arrays, objects.
ఉదా: Boolean(123) → true, Boolean(0) → false, Boolean("hello") → true, Boolean("") → false.

📋 6. Summary & Interview-Ready Cheatsheet

Built-in conversion functions:

  • String(value) → converts to string (creates copy).
  • Number(value) → converts to number; NaN if invalid.
  • parseInt(string) → parses integer from start until non-digit.
  • Boolean(value) → returns true/false based on truthy/falsy.

Common conversions:

// Number to String String(123) // "123" // Boolean to String String(true) // "true" // String to Number Number("123") // 123 Number("123abc") // NaN parseInt("123abc")// 123 // Boolean to Number Number(true) // 1 Number(false) // 0 // to Boolean Boolean(1) // true Boolean(0) // false Boolean("Hello") // true Boolean("") // false
🧠 Kalyan's note: In technical MCQ rounds, expect questions like "What is the output of Boolean('0')?" → Answer: true (because non-empty string). Or "parseInt('10abc')" → 10. Practice these daily.

Understanding type conversion prevents bugs and is essential for form validation, API handling, and logical conditions in projects.

📌 తెలుగు సారాంశం :
• String() → దేనినైనా string గా మారుస్తుంది.
• Number() → number గా మారుస్తుంది; invalid అయితే NaN.
• parseInt() → left నుండి digits తీసుకుంటుంది, non-digit వద్ద ఆగుతుంది.
• Boolean() → truthy/falsy బట్టి true/false ఇస్తుంది.
• Falsy values: 0, NaN, "", null, undefined, false.
• Truthy values: మిగిలినవన్నీ. ఈ concepts ఇంటర్వ్యూలలో చాలా ముఖ్యమైనవి. ప్రాక్టీస్ చేయండి!
