📌 1. The Problem: Storing Multiple Values
❌ Without arrays, storing 5 names requires 5 different variables:
let a = "Kalyan"; let b = "Mega"; let c = "Shilpa"; let d = "Somesh"; let e = "Divya";

Problems with this approach:

  • Variable explosion: 1000 names → 1000 variables (hard to create and manage)
  • Access difficulty: To get the 234th person, you must remember the variable name — impossible at scale!
💡 Solution: Arrays — store all values in a single variable and access them by index (position).

📌 తెలుగు : Arrays లేకుండా, 5 names ని store చేయడానికి 5 variables కావాలి. 1000 names కి 1000 variables → impossible. Access చేయడం కూడా కష్టం. Arrays ద్వారా ఒకే variable లో అన్ని values ని store చేసి, index (position) ద్వారా access చేయవచ్చు.

📦 2. What is an Array? Syntax & Memory

An array is a collection of values stored in sequential order, accessed using a single variable name and indexes (starting from 0).

let arr = ["Kalyan", "Mega", "Shilpa", "Somesh", "Divya"]; // Index: 0 1 2 3 4

Memory view:

  • Array is an object stored in Heap.
  • Each value occupies a cell with a numeric index (0, 1, 2...).
  • The variable (arr) stores a reference (address) to the array object in Stack.
  • Access: arr[0] (first element), arr[1] (second), etc.
✅ Arrays can hold any data type (numbers, strings, booleans, objects, even other arrays). Elements don't have to be the same type.

📌 తెలుగు : Array అనేది sequential order లో values ని store చేసే collection. ఒకే variable name, indexes (0 నుండి start) ద్వారా access. Array ఒక object, Heap లో store అవుతుంది. Variable Stack లో reference (address) store చేస్తుంది. Arrays ఏ data type ని అయినా store చేయగలవు.

🛠️ 3. Creating Arrays: Literal vs Constructor
// Method 1: Array Literal (most common) let fruits = ["Apple", "Banana", "Mango"]; // Method 2: Array Constructor (new Array()) let numbers = new Array(5); // creates empty array of length 5 numbers[0] = 10; numbers[1] = 20; numbers[2] = 30; numbers[3] = 40; numbers[4] = 50;
🔥 Best Practice: Use array literal [] for simplicity. Constructor is useful when you need a fixed-size empty array.

Arrays can hold mixed data types:

let mixed = ["Kalyan", 5.9, 68, true, "Tap Academy"]; console.log(mixed[0]); // "Kalyan" console.log(mixed[3]); // true

📌 తెలుగు : Arrays ని create చేయడానికి రెండు మార్గాలు: Array Literal ([]) — సాధారణ, మరియు Array Constructor (new Array()) — fixed-size empty array కోసం. Best practice: literal ని వాడండి. Arrays mixed data types ని కూడా store చేయగలవు.

🔍 4. Accessing Elements & length Property
let arr = [10, 20, 30, 40, 50]; console.log(arr[0]); // 10 (first element) console.log(arr[2]); // 30 console.log(arr[4]); // 50 (last element) console.log(arr.length); // 5 (number of elements) console.log(arr[arr.length - 1]); // 50 (last element dynamically)
💡 length property returns the number of elements. Last index is always length - 1.

Every array object has built-in properties: length (and methods like push, pop, map, filter etc.).

📌 తెలుగు : Array elements ని arr[index] తో access చేస్తాం. length property మొత్తం elements count ని ఇస్తుంది. Last index ఎప్పుడూ length - 1. ప్రతీ array కి length property మరియు methods (push, pop, map, filter) ఉంటాయి.

🔄 5. Iterating with for Loop (Forward, Reverse, Dynamic length)
let arr = [10, 20, 30, 40, 50]; // Forward iteration for (let i = 0; i < arr.length; i++) { console.log(arr[i]); // 10,20,30,40,50 } // Reverse iteration for (let i = arr.length - 1; i >= 0; i--) { console.log(arr[i]); // 50,40,30,20,10 }
🔥 Why loops with arrays? Without loops, printing 1000 elements would require 1000 lines of code. Loops make it scalable.

Using arr.length dynamically adapts to array size — change array size, loop still works!

📌 తెలుగు : Arrays ని iterate చేయడానికి for loop వాడతాం. Forward: for (let i=0; i. Reverse: for (let i=arr.length-1; i>=0; i--). arr.length వలన loop dynamic గా work చేస్తుంది — array size మారినా loop adjust అవుతుంది.

✨ 6. for...of Loop (Simpler, Element-focused)
let arr = [10, 20, 30, 40, 50]; // for...of - gives values directly for (let element of arr) { console.log(element); // 10,20,30,40,50 }
✅ Advantages: No need to manage index, condition, or increment. Automatically goes from first to last element.

Limitation: You cannot control direction (only forward), cannot skip elements, cannot use index for complex logic. Use traditional for loop when you need index control.

📌 తెలుగు : for...of loop values ని నేరుగా ఇస్తుంది — index, condition, increment అవసరం లేదు. మంచి సింపుల్ iteration. కానీ direction మార్చలేరు (forward only), skip చేయలేరు. Index control కావాలంటే traditional for loop వాడండి.

📖 7. Summary: Arrays at a Glance
  • Why Arrays? Store multiple values in one variable, access by index (position).
  • Syntax: let arr = [value1, value2, ...];
  • Indexing: Starts at 0 (first element), ends at length - 1.
  • length property: arr.length gives number of elements.
  • Iteration: for loop (full control), for...of (values only).
  • Arrays are objects → stored in Heap, reference in Stack.
// Quick recap example let colors = ["Red", "Green", "Blue"]; console.log(colors[0]); // Red console.log(colors.length); // 3 for (let color of colors) { console.log(color); // Red, Green, Blue }
