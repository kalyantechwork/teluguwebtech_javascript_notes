📌 1. The Challenge: console.log() adds newline

We want to print 5 stars side by side (horizontally): * * * * *. But console.log("*") always moves cursor to the next line after printing. So multiple console.log statements print stars vertically.

console.log("*"); // prints * and newline console.log("*"); // next line // Output: // * // *

Solution: Build the entire row as a string first, then print it once using console.log.

📌 తెలుగు : మనం 5 stars ని ఒకే లైన్ లో (horizontal గా) print చేయాలి: * * * * *. console.log("*") ప్రతిసారీ new line వేస్తుంది. కాబట్టి ఒక్కో star ని వేర్వేరుగా print చేస్తే vertical గా వస్తాయి. పరిష్కారం: మొత్తం row ని ఒక string గా build చేసి, ఒక్కసారి print చేయడం.

🔧 2. Building a Row: String Concatenation

We create an empty string variable row = "", then keep adding stars to it using concatenation (+).

let row = ""; // empty string row = row + "*"; // row becomes "*" row = row + "*"; // row becomes "**" row = row + "*"; // "***" row = row + "*"; // "****" row = row + "*"; // "*****" console.log(row); // prints ***** horizontally

Instead of repeating the line 5 times, we use a loop to repeat the concatenation.

let row = ""; for (let j = 1; j <= 5; j++) { row = row + "*"; } console.log(row); // *****
✅ This gives us one row with 5 stars side by side.

📌 తెలుగు : ఖాళీ string row = "" తీసుకొని, దానికి stars ని concatenate (జోడించి) build చేస్తాం. ఇలా 5 సార్లు చేయడానికి loop ఉపయోగిస్తాం. ఫలితంగా ఒకే లైన్ లో 5 stars వస్తాయి.

📐 3. From One Row to Multiple Rows: Nested Loops

To print a square pattern of 5 rows and 5 columns, we need to repeat the row-building process 5 times (once for each row). This is achieved by placing the entire row-building loop inside another loop.

let n = 5; for (let i = 1; i <= n; i++) { // Outer loop: rows let row = ""; for (let j = 1; j <= n; j++) { // Inner loop: columns in current row row = row + "* "; } console.log(row); // print the completed row }
* * * * *
* * * * *
* * * * *
* * * * *
* * * * *
🔥 Nested Loop Logic: Outer loop (i) controls rows. For each row, inner loop (j) builds the row string by adding stars, then we print it.

We can make the pattern dynamic by using a variable n:

let n = 8; // change to any number for (let i = 1; i <= n; i++) { let row = ""; for (let j = 1; j <= n; j++) { row = row + "* "; } console.log(row); }

📌 తెలుగు : Square pattern కోసం (5 rows, 5 columns), row-building process ని 5 సార్లు repeat చేయాలి. దీనికోసం row-building loop ని మరొక loop లోపల ఉంచుతాం. Outer loop (i) rows ని నియంత్రిస్తుంది. Inner loop (j) ప్రతీ row లో columns ని build చేస్తుంది. n variable వలన pattern dynamic గా మారుతుంది.

🧩 4. Outer Loop (Rows) vs Inner Loop (Columns)

Let's trace the execution for n = 3:

// i=1 (first row) // j=1 → row = "*" // j=2 → row = "**" // j=3 → row = "***" // print row → "***" // i=2 (second row) // j=1,2,3 → build "***", print // i=3 (third row) // j=1,2,3 → build "***", print
💡 Key Insight: The outer loop runs n times. Each time the outer loop runs, the inner loop runs completely n times. Total iterations = n × n.

We use different loop variables: i for outer (rows), j for inner (columns) to avoid confusion.

📌 తెలుగు : Outer loop (i) n సార్లు నడుస్తుంది (rows). ప్రతీసారీ inner loop (j) పూర్తిగా n సార్లు నడుస్తుంది (columns). మొత్తం iterations = n × n. Rows కి i, columns కి j అనే వేర్వేరు variables వాడతాం.

⚙️ 5. Scalable Pattern using Variable 'n'

Instead of hardcoding 5, use a variable n. Change one place to change the entire pattern size.

let n = 10; // 10x10 square for (let i = 1; i <= n; i++) { let row = ""; for (let j = 1; j <= n; j++) { row = row + "* "; } console.log(row); }

Output for n=4:

* * * *
* * * *
* * * *
✅ This is the base pattern. From here, we can create variations: hollow squares, triangles, pyramids, number patterns, etc. by modifying the inner loop's condition or what we add to the row.

📌 తెలుగు : Hardcode చేయకుండా n variable ఉపయోగించండి. దీనివలన ఒకేచోట n ని మారిస్తే pattern size మారుతుంది. ఇది base pattern. ఇక్కడ నుండి variations (hollow squares, triangles, pyramids) సృష్టించవచ్చు.

⚠️ 6. Common Mistakes & Tips
  • Mistake 1: Forgetting to re-initialize row = "" inside the outer loop. If you declare row outside, it keeps growing for each row → wrong output.
  • Mistake 2: Using same variable for both loops (e.g., using i for both). Always use different variables (i for outer, j for inner).
  • Mistake 3: Off-by-one errors: i <= n vs i < n. Both work if indices are adjusted, but be consistent.
// Correct way - row re-initialized for each row for (let i = 1; i <= n; i++) { let row = ""; // fresh empty string for each new row for (let j = 1; j <= n; j++) { row += "* "; } console.log(row); }
🧠 Kalyan's Advice: Practice this base pattern until it becomes muscle memory. Then try variations: right-angled triangle, inverted triangle, pyramid, diamond. Pattern programming is the best way to master nested loops.

📌 తెలుగు : సాధారణ తప్పులు: 1) ప్రతీ row కి row = "" re-initialize చేయకపోవడం. 2) రెండు loops కి ఒకే variable వాడడం (i, j వేర్వేరుగా వాడండి). 3) Off-by-one errors. ఈ base pattern ని బాగా ప్రాక్టీస్ చేయండి. తర్వాత triangles, pyramids వంటి variations ని solve చేయండి.

🚀 7. What's Next? Pattern Variations

Once you understand the square pattern, you can modify it to create:

  • Hollow square — stars only on borders, spaces inside (using if-else inside inner loop)
  • Right-angled triangle — j runs from 1 to i (instead of n)
  • Inverted triangle — j runs from 1 to n-i+1
  • Pyramid / Diamond — combination of spaces and stars
  • Number patterns — print i or j instead of star
// Example: Right-angled triangle (n=5) // * // * * // * * * // * * * * // * * * * * for (let i = 1; i <= n; i++) { let row = ""; for (let j = 1; j <= i; j++) { // j goes up to i row = row + "* "; } console.log(row); }
