📌 1. Why Control Structures? Beyond Sequential Execution

So far, our programs executed sequentially — line by line from top to bottom. But real-world logic often requires decision making and repetition. For example:

  • ✅ If age >= 18, allow entry to club; else deny.
  • ✅ If cart total >= ₹10,000, offer free shipping; else charge shipping.
  • ✅ If VIP, give VIP lounge; else regular lounge.
  • ✅ Repeat instructions 5 times (loops).
💡 Control Structures allow us to alter the flow of execution — either by making decisions (conditionals) or repeating blocks (loops).

Two main categories: Conditional Statements (if, if-else, else-if, switch, ternary) and Loops (for, while, do-while, for-in, for-of). In this class, we focus on conditionals.

📌 తెలుగు : ఇప్పటివరకు మన programs sequential గా execute అయ్యాయి (పై నుండి కిందికి line by line). కానీ real-world లో నిర్ణయాలు తీసుకోవడం (decision making) మరియు పునరావృతం (repetition) అవసరం. ఉదా: వయస్సు >= 18 అయితే క్లబ్ లోకి అనుమతించు, లేకపోతే నిరాకరించు. దీనికి Control Structures ఉపయోగిస్తాం. ఇందులో రెండు రకాలు: Conditional Statements (if, else, switch) మరియు Loops (for, while).

🔹 2. if Statement — Execute block only if condition is true

The if statement executes a block of code only if the specified condition evaluates to true. If false, the block is skipped.

let age = 20; if (age >= 18) { console.log("Welcome to the club!"); } // Output: Welcome to the club!

Real-world e-commerce example (Free shipping):

let cartTotal = 12000; // ₹12,000 if (cartTotal >= 10000) { console.log("You have qualified for FREE shipping!"); } // Output: You have qualified for FREE shipping!

If cartTotal = 2000, nothing prints because condition is false.

✅ Syntax: if (condition) { // code to run when true }

📌 తెలుగు : if statement: condition true అయితే మాత్రమే block execute అవుతుంది. False అయితే skip. ఉదా: if (age >= 18) { console.log("Welcome"); }. E-commerce example: cartTotal >= 10000 అయితే free shipping message చూపించు.

🔹 3. if-else Statement — One path or the other

When you have two alternatives: if condition true, execute block1; otherwise (else) execute block2.

let isVIP = true; if (isVIP === true) { console.log("Welcome to VIP Lounge!"); } else { console.log("Welcome to Regular Lounge."); } // Output: Welcome to VIP Lounge!

ATM / Bank withdrawal example:

let balance = 1000; let withdrawAmount = 700; if (balance >= withdrawAmount) { console.log("Transaction Successful! Please collect cash."); } else { console.log("Insufficient Balance. Transaction Failed."); } // Output: Transaction Successful! (since 1000 >= 700)

If withdrawAmount = 1500, output becomes "Insufficient Balance".

📌 తెలుగు : if-else: రెండు paths. Condition true అయితే if block, false అయితే else block execute అవుతుంది. ఉదా: VIP lounge vs Regular lounge. Bank withdrawal: balance >= withdrawAmount అయితే "Transaction Successful", లేకపోతే "Insufficient Balance".

🔹 4. if-else if-else Ladder — Multiple conditions (Railway ticket example)

When you have more than two possible paths, use else if to check additional conditions. The first condition that is true executes its block, then exits the ladder.

┌─────────────────────────┐
│ if (age < 12) → child │
│ else if (age >= 65) → senior │
│ else → adult │
└─────────────────────────┘
let age = 14; // Try changing: 8, 14, 70 if (age < 12) { console.log("Child Ticket Price: ₹150"); } else if (age >= 65) { console.log("Senior Citizen Ticket Price: ₹200"); } else { console.log("Adult Ticket Price: ₹300"); } // Output: Adult Ticket Price: ₹300

Explanation: age=14 is not <12, not >=65, so falls to else → adult price.

// Try different ages: // age = 8 → Child Ticket Price // age = 70 → Senior Citizen Ticket Price // age = 30 → Adult Ticket Price
🔥 Important: Only one block executes — the first true condition. The rest are ignored.

📌 తెలుగు : if-else if-else ladder: మూడు లేదా అంతకంటే ఎక్కువ options ఉన్నప్పుడు వాడతాం. Railway ticket example: age < 12 → child price, age >= 65 → senior price, else → adult price. First true condition మాత్రమే execute అవుతుంది.

📚 5. Nested Conditions & Flow Visualization

We can also place an if or if-else inside another if block — this is called nesting. Useful for complex decision trees.

let age = 50; let hasID = true; if (age >= 18) { if (hasID === true) { console.log("Valid ID. Entry granted."); } else { console.log("ID required. Entry denied."); } } else { console.log("Underage. Entry denied."); } // Output: Valid ID. Entry granted.

Remember: Control structures can be combined with logical operators (&&, ||) to write concise conditions.

📖 6. Summary & Quick Reference

Conditional Statements in JavaScript:

  • if (condition) { } → execute only if true
  • if (condition) { } else { } → one of two paths
  • if (cond1) { } else if (cond2) { } else { } → multiple paths
  • Conditions use comparison operators (==, ===, >, <, >=, <=, !=, !==)
  • Braces { } define code blocks. Indentation improves readability.
// Simple if let temperature = 35; if (temperature > 30) { console.log("It's a hot day!"); } // if-else let marks = 35; if (marks >= 40) { console.log("Pass"); } else { console.log("Fail"); } // else-if ladder let score = 85; if (score >= 90) { console.log("Grade A"); } else if (score >= 75) { console.log("Grade B"); } else { console.log("Grade C"); }
