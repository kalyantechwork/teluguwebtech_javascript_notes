📌 1. Recap: Simple Square Pattern (Nested Loops)

In the previous class, we created a solid square pattern using nested loops:

let n = 5; for (let i = 1; i <= n; i++) { let row = ""; for (let j = 1; j <= n; j++) { row += "* "; } console.log(row); } // Output: 5x5 solid star square

Now we want to create a hollow square — stars only on borders, spaces inside.

📌 తెలుగు : మునుపటి class లో solid square pattern ని nested loops తో create చేశాం. ఇప్పుడు hollow square create చేయాలి — borders మీద మాత్రమే stars, లోపల spaces ఉండాలి.

🔲 2. Hollow Square: Stars on Borders Only

Conditions for printing star (instead of space):

  • First row (i === 1)
  • Last row (i === n)
  • First column (j === 1)
  • Last column (j === n)

If any condition is true → print star, else print space.

let n = 8; for (let i = 1; i <= n; i++) { let row = ""; for (let j = 1; j <= n; j++) { if (i === 1 || i === n || j === 1 || j === n) { row += "* "; } else { row += " "; // two spaces for alignment } } console.log(row); }
💡 Note: Using n dynamically makes the pattern scalable. Change n to 10, and you get a 10x10 hollow square.

We can also print row numbers inside instead of stars:

// Replace "*" with i (row number) if (i === 1 || i === n || j === 1 || j === n) { row += i + " "; } else { row += " "; }

📌 తెలుగు : Hollow square కోసం conditions: first row (i===1), last row (i===n), first column (j===1), last column (j===n). ఏదైనా condition true అయితే star print చేయాలి, లేకపోతే space. n ని dynamic గా ఉంచడం వలన pattern scalable అవుతుంది. Stars స్థానంలో row numbers (i) ని కూడా print చేయవచ్చు.

🔢 3. Zero-based Indexing (0 to n-1)

We can also write loops starting from 0 instead of 1. Then conditions change: first row = 0, last row = n-1.

let n = 11; for (let i = 0; i < n; i++) { // i from 0 to 10 let row = ""; for (let j = 0; j < n; j++) { // j from 0 to 10 if (i === 0 || i === n-1 || j === 0 || j === n-1) { row += "* "; } else { row += " "; } } console.log(row); }

Both approaches work. Choose whichever is comfortable. In competitive programming, zero-based is common.

📌 తెలుగు : Loops ని 0 నుండి కూడా start చేయవచ్చు (i = 0 to n-1). అప్పుడు conditions: first row i===0, last row i===n-1, first column j===0, last column j===n-1. రెండు approaches సరైనవే.

🎯 4. Adding Center Lines: Vertical & Horizontal

To draw center lines, we need the middle index. For odd n (e.g., 11), middle is Math.floor(n/2).

let n = 11; let center = Math.floor(n / 2); // 5 (when n=11, 0-based center is 5) for (let i = 0; i < n; i++) { let row = ""; for (let j = 0; j < n; j++) { if (i === 0 || i === n-1 || j === 0 || j === n-1 || i === center || j === center) { row += "* "; } else { row += " "; } } console.log(row); }
🔥 Important: In JavaScript, division of integers returns float. Use Math.floor(n/2) for integer division.

Without Math.floor, 11/2 = 5.5 — no column index matches 5.5, so the line wouldn't appear!

📌 తెలుగు : Center lines కోసం middle index కావాలి: Math.floor(n/2). JavaScript లో integer division ఫలితం float వస్తుంది (11/2 = 5.5). కాబట్టి Math.floor() తో integer గా మార్చాలి. లేకపోతే center line print అవ్వదు.

📐 5. Main Diagonal: i === j

For the main diagonal (top-left to bottom-right), the condition is i === j.

let n = 11; let center = Math.floor(n/2); for (let i = 0; i < n; i++) { let row = ""; for (let j = 0; j < n; j++) { if (i === 0 || i === n-1 || j === 0 || j === n-1 || i === center || j === center || i === j) { row += "* "; } else { row += " "; } } console.log(row); }

📌 తెలుగు : Main diagonal (top-left to bottom-right) కోసం condition: i === j. i మరియు j విలువలు సమానమైనచోట star వస్తుంది.

📐 6. Anti-Diagonal: i + j === n-1

For the anti-diagonal (top-right to bottom-left), condition is i + j === n - 1.

// Adding anti-diagonal if (i === 0 || i === n-1 || j === 0 || j === n-1 || i === center || j === center || i === j || (i + j === n-1)) { row += "* "; }
✅ For n=11, n-1 = 10. So i + j = 10 gives the anti-diagonal.

📌 తెలుగు : Anti-diagonal (top-right to bottom-left) కోసం condition: i + j === n-1. n=11 అయితే, n-1=10. కాబట్టి i + j = 10 ఉన్నచోట stars వస్తాయి.

🧩 7. Advanced Diagonals: Center Intersecting Lines

We can create more lines passing through the center:

  • i + j = center (from top-left to center)
  • i - j = center (from bottom-left to center)
  • j - i = center (from top-right to center)
  • i + j = (n-1) + center (from bottom-right to center)
let n = 15; let center = Math.floor(n/2); for (let i = 0; i < n; i++) { let row = ""; for (let j = 0; j < n; j++) { if (i === 0 || i === n-1 || j === 0 || j === n-1 || i === center || j === center || i === j || (i + j === n-1) || (i + j === center) || (i - j === center) || (j - i === center) || (i + j === (n-1) + center)) { row += "* "; } else { row += " "; } } console.log(row); }
🧠 Kalyan's Note: Pattern programming builds logical thinking. Solve at least 200-250 pattern problems to master loops and conditionals — this is the foundation for cracking coding interviews.

📌 తెలుగు : Center గుండా పోయే lines కోసం conditions:
• i + j = center
• i - j = center
• j - i = center
• i + j = (n-1) + center
ఇలా conditions కలుపుతూ పోతే complex patterns వస్తాయి. Pattern programming వలన logical thinking పెరుగుతుంది — ఇది coding interviews లో చాలా ఉపయోగపడుతుంది.

📖 8. Summary & Next Steps

Key takeaways from pattern programming:

  • Loops + Conditionals = Patterns
  • Hollow patterns require border conditions (first/last row, first/last column)
  • Diagonals use i === j and i + j === n-1
  • Center lines need Math.floor(n/2) for odd n
  • Always use Math.floor() for integer division when indices are involved
🎯 Pattern programming is not just about stars — it builds the logical muscle required for algorithms, data structures, and real-world problem solving. Keep practicing!
