📑 Shallow Copy vs Deep Copy in JavaScript

📦 Spread operator (...), slice(), structuredClone() | Nested arrays & objects | Memory references — by Kalyan (Telugu WebTech)
📌 1. Creating a Copy: spread (...) and slice()
let arr1 = [10, 20, 30, 40]; // Method 1: Spread operator (shallow copy) let arr2 = [...arr1]; // Method 2: slice() method (shallow copy) let arr3 = arr1.slice(); arr1.push(99); console.log(arr1); // [10,20,30,40,99] console.log(arr2); // [10,20,30,40] (unchanged) console.log(arr3); // [10,20,30,40] (unchanged)
✅ Both [...arr] and arr.slice() create a shallow copy. For primitive values (numbers, strings), modifying the original does NOT affect the copy.

📌 తెలుగు : Array ని copy చేయడానికి రెండు మార్గాలు: spread operator ([...arr]) మరియు slice(). ఇవి shallow copy ని create చేస్తాయి. Primitive values (numbers, strings) విషయంలో, original ని modify చేస్తే copy మారదు.

⚠️ 2. The Problem: Shallow Copy with Nested Arrays
let arr1 = [[10, 20], [30, 40]]; let arr2 = arr1.slice(); // or [...arr1] // Modify nested array in original arr1[0].push(99); console.log(arr1); // [[10,20,99], [30,40]] console.log(arr2); // [[10,20,99], [30,40]] ← ALSO CHANGED!
⚠️ Why? Shallow copy copies only the references (addresses) to inner arrays, not the inner arrays themselves. Both arr1[0] and arr2[0] point to the SAME inner array.
🔥 Shallow Copy = only top-level properties are copied. Nested objects/arrays are shared by reference.

📌 తెలుగు : Nested arrays లో shallow copy problem: Shallow copy inner arrays యొక్క references (addresses) ని మాత్రమే copy చేస్తుంది, arrays ని కాదు. కాబట్టి original nested array ని modify చేస్తే, copy లో కూడా అదే మార్పు కనిపిస్తుంది.

📦 3. Shallow Copy with Objects in Array
let arr1 = [{ name: "Ann" }, { name: "Ben" }]; let arr2 = arr1.slice(); // shallow copy // Modify object property in original arr1[0].name = "Kalyan"; console.log(arr1); // [{ name: "Kalyan" }, { name: "Ben" }] console.log(arr2); // [{ name: "Kalyan" }, { name: "Ben" }] ← ALSO CHANGED!
💡 Same behavior: Objects inside arrays are also stored by reference. Shallow copy copies the references, so both arrays point to the same objects.

📌 తెలుగు : Objects కూడా arrays లాగే references ద్వారా store అవుతాయి. Shallow copy references ని copy చేస్తుంది, కాబట్టి original object ని modify చేస్తే copy లో కూడా మార్పు కనిపిస్తుంది.

✅ 4. Deep Copy using structuredClone()
let arr1 = [[10, 20], [30, 40]]; let arr2 = structuredClone(arr1); // Deep copy arr1[0].push(99); console.log(arr1); // [[10,20,99], [30,40]] console.log(arr2); // [[10,20], [30,40]] ← UNCHANGED! (true copy)
✅ structuredClone() creates a deep copy — it recursively copies all nested objects/arrays. The copy is completely independent.
// Deep copy with objects let objArr = [{ name: "Ann" }, { name: "Ben" }]; let deepCopy = structuredClone(objArr); objArr[0].name = "Kalyan"; console.log(objArr); // [{ name: "Kalyan" }, { name: "Ben" }] console.log(deepCopy); // [{ name: "Ann" }, { name: "Ben" }] (unchanged)

📌 తెలుగు : structuredClone() deep copy ని create చేస్తుంది — ఇది అన్ని nested arrays/objects ని recursively copy చేస్తుంది. Copy పూర్తిగా independent గా ఉంటుంది. Original modify చేస్తే copy మారదు.

🧠 5. Memory Visualization: Shallow vs Deep Copy
// Shallow Copy (spread / slice) let original = [[10, 20], [30, 40]]; let shallow = [...original]; // Memory: // original → [ref1, ref2] (stack) // shallow → [ref1, ref2] (same references!) // ref1 points to [10,20] in Heap // ref2 points to [30,40] in Heap // Deep Copy (structuredClone) let deep = structuredClone(original); // Memory: // original → [ref1, ref2] // deep → [ref3, ref4] (new references!) // ref3 points to NEW [10,20] copy // ref4 points to NEW [30,40] copy
🔥 Key Insight: Shallow copy = copy references. Deep copy = create new copies of all nested structures.

📌 తెలుగు : Shallow copy: references మాత్రమే copy అవుతాయి. Deep copy: అన్ని nested structures యొక్క కొత్త copies create అవుతాయి, పూర్తి independence.

📖 6. Summary: Shallow vs Deep Copy
  • Shallow Copy methods: spread operator ([...arr]), arr.slice(), Array.from(arr), Object.assign([], arr)
  • Deep Copy method: structuredClone(arr) (modern, recommended)
  • When to use shallow copy: For flat arrays with primitive values only (numbers, strings, booleans).
  • When to use deep copy: For nested arrays/objects where you need complete independence.
  • Alternative deep copy (older): JSON.parse(JSON.stringify(arr)) (has limitations: cannot copy functions, undefined, etc.)
// Quick reference let flat = [1, 2, 3]; let flatCopy = [...flat]; // shallow, but fine (primitives) let nested = [[1, 2], [3, 4]]; let nestedShallow = [...nested]; // PROBLEM: shares inner arrays let nestedDeep = structuredClone(nested); // SOLUTION: independent copy
🧠 Kalyan's Advice: Always understand your data structure. For nested arrays/objects, prefer structuredClone() to avoid accidental shared mutations.

📌 తెలుగు సారాంశం :
• Shallow Copy methods: spread, slice, Array.from — references copy అవుతాయి.
• Deep Copy method: structuredClone() — నిజమైన independent copy.
• Use shallow copy: flat arrays (primitives only).
• Use deep copy: nested arrays/objects.
• Alternate (old): JSON.parse(JSON.stringify()) — limitations ఉన్నాయి.
Nested data కోసం structuredClone() ని వాడండి.
