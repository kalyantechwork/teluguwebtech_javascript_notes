🎯 Functions as First-Class Citizens in JavaScript

📦 Functions are Objects | Store in variable | Pass as argument (Higher-Order Functions, Callbacks) | Return from function | Memory (Stack & Heap) — by Kalyan (Telugu WebTech)
📌 1. What is a First-Class Citizen?

A value in JavaScript is called a first-class citizen if it can do these three things:

① Stored in a variable
(assigned to a variable)
② Passed as argument
to a function
③ Returned from a function
as a value
🔥 Objects in JavaScript are first-class citizens. They satisfy all three conditions.

In this class, we prove that functions are also objects and therefore first-class citizens in JavaScript.

📌 తెలుగు : ఒక value ని first-class citizen అంటే అది మూడు పనులు చేయగలిగితే:
① variable లో store చేయగలగడం
② function కి argument గా pass చేయగలగడం
③ function నుండి return చేయగలగడం
JavaScript లో objects first-class citizens. ఇప్పుడు functions కూడా objects అని prove చేద్దాం.

📦 2. Objects are First-Class Citizens (Quick Proof)
// 1. Store object in a variable let car = { name: "BMW", cost: 75.6, mileage: 8.9 }; // 2. Pass object as argument to a function function showCar(obj) { console.log(obj); } showCar(car); // Object passed as argument // 3. Return object from a function function getCar() { return { name: "Audi", cost: 85.0 }; } let newCar = getCar(); // Object returned from function
✅ Objects satisfy all three conditions → First-Class Citizens in JavaScript.

Memory: Object is created in Heap, variable stores reference (address) in Stack.

📌 తెలుగు : Objects మూడు conditions ని satisfy చేస్తాయి:
① variable లో store చేయవచ్చు, ② function కి argument గా pass చేయవచ్చు, ③ function నుండి return చేయవచ్చు. Object Heap లో create అవుతుంది, variable Stack లో reference (address) store చేస్తుంది.

⚡ 3. Proof: Functions are Objects in JavaScript

Just like objects, functions have properties (name, length) and methods (call, apply, bind). This proves functions are also objects.

function add(a, b) { console.log(a + b); } // Functions have properties (like objects) console.log(add.name); // "add" console.log(add.length); // 2 (number of parameters) // Functions have methods (call, apply, bind) console.log(typeof add.call); // "function" console.log(typeof add.apply); // "function"
🔥 Memory: When a function is created, JavaScript creates a function object in Heap with:
  • Properties: name, length, etc.
  • Methods: call, apply, bind
The function name (add) is a reference variable storing the address of this function object.

📌 తెలుగు : Functions కి properties (name, length) మరియు methods (call, apply, bind) ఉంటాయి — ఇది functions కూడా objects అని రుజుపు చేస్తుంది. Function create అయినప్పుడు Heap లో function object create అవుతుంది. Function name (add) అనేది ఆ object కి reference.

📝 4. Storing Functions in Variables (Multiple References)

Since functions are objects, they can have multiple references. We can assign the same function to different variables.

function add(a, b) { return a + b; } let addRef1 = add; // Both reference the SAME function object let addRef2 = add; console.log(addRef1(10, 20)); // 30 console.log(addRef2(100, 200)); // 300
💡 Multiple references to the same function object: Just like multiple keys to the same locker. The function object is ONE in Heap, referenced by multiple variables.

📌 తెలుగు : Functions objects కాబట్టి వాటికి multiple references ఉండవచ్చు. ఒకే function ని వివిధ variables కి assign చేయవచ్చు. అన్ని variables ఒకే Heap object ని point చేస్తాయి. ఇది pass by reference కి ఉదాహరణ.

🔄 5. Passing Function as Argument (Higher-Order Function & Callback)

A function that takes another function as an argument is called a Higher-Order Function. The function passed as argument is called a Callback Function.

function add(a, b) { return a + b; } // Higher-Order Function (receives a function as argument) function executeOperation(operation, x, y) { return operation(x, y); // calling the callback } let result = executeOperation(add, 100, 200); console.log(result); // 300
🔥 Memory: The reference (address) of the add function is passed to operation parameter. Both point to the same function object in Heap.

📌 తెలుగు : Higher-Order Function: మరొక function ని argument గా తీసుకునే function. Callback Function: argument గా pass చేయబడే function. Reference (address) pass అవుతుంది — ఇది pass by reference కు ఉదాహరణ.

📤 6. Returning Function from Function

A function can return another function. This is the third criteria for first-class citizenship.

function createMultiplier(multiplier) { return function(number) { return number * multiplier; }; } let double = createMultiplier(2); let triple = createMultiplier(3); console.log(double(5)); // 10 console.log(triple(5)); // 15
✅ Functions can be returned from other functions → this leads to powerful concepts like Closures (next class).

📌 తెలుగు : ఒక function మరొక function ని return చేయగలదు. ఇది first-class citizenship కి మూడవ నియమాన్ని సంతృప్తి పరుస్తుంది. ఇది Closures వంటి powerful concepts కి దారి తీస్తుంది.

📖 7. Summary: Functions as First-Class Citizens
  • First-Class Citizen = can be stored in variable, passed as argument, returned from function.
  • Objects are first-class citizens in JavaScript.
  • Functions are also objects (they have properties like name, length and methods like call, apply, bind).
  • Higher-Order Function: A function that takes another function as argument OR returns a function.
  • Callback Function: A function passed as argument to another function.
// Quick recap - all three abilities // 1. Store in variable let greet = function() { console.log("Hello"); }; // 2. Pass as argument function run(fn) { fn(); } run(greet); // Hello // 3. Return from function function getGreeter() { return function() { console.log("Hi"); }; } let sayHi = getGreeter(); sayHi(); // Hi
🧠 Kalyan's Advice: Understanding that functions are objects is the key to mastering JavaScript. This concept enables callbacks, promises, async/await, and functional programming patterns.

📌 తెలుగు సారాంశం :
• First-Class Citizen: variable లో store, argument గా pass, function నుండి return చేయగలిగేది.
• Objects first-class citizens.
• Functions కూడా objects (properties: name, length; methods: call, apply, bind).
• Higher-Order Function: function ని argument గా తీసుకునే లేదా function ని return చేసే function.
• Callback Function: argument గా pass చేయబడే function.
Functions are objects అనే concept ని అర్థం చేసుకోవడం JavaScript లో callbacks, promises, async/await వంటి advanced concepts ని నేర్చుకోవడానికి మూలస్తంభం.
